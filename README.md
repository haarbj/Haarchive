# The Haarchive

A Next.js + TypeScript + Tailwind CSS website for a long-term educational resource on distance running, and the home of "The Onus to Quit: Training the Body, Strengthening the Mind."

## Development

\`\`\`bash
npm install
npm run dev
\`\`\`

## Checks

\`\`\`bash
npm run lint
npm run build
\`\`\`
