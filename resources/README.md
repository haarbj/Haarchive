# Resources

This directory contains the reference material used to build the Haarchive.

These resources are references, not content to reproduce.

When editing the website:

- Read all relevant resources before making changes.
- Synthesize information across multiple sources.
- Do not treat any single book as the source of truth.
- Prioritize the strongest available evidence.
- Maintain a single, consistent writing style.
- Avoid duplicate information.
- Expand existing pages whenever appropriate.
- Create new pages only when a concept deserves its own article.
- Cross-link related concepts throughout the site.

The goal is to build one cohesive educational resource, not a collection of book summaries.
